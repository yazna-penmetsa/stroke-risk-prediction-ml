# Stroke Risk Prediction using Machine Learning

This project analyzes stroke and cardiovascular disease risk factors using health survey data and applies machine learning models to predict stroke occurrence.

---

## Dataset
- Source: BRFSS / Kaggle cardiovascular dataset
- Variables include age, gender, BMI, hypertension, glucose level, smoking, alcohol intake, and physical activity

---

## Methods
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical analysis and correlation assessment
- Machine learning models:
  - Logistic Regression
  - K-Nearest Neighbors (KNN)
  - Random Forest
